1. あなたのせいじゃない！「意志力」という幻想からの解放

多くの人が「習慣が続かないのは、自分の意志が弱いからだ」と思い込んでいます。しかし、科学的な結論は真逆です。習慣化とは、意志力の問題ではなく、技術の問題です。

私たちの脳は、変化を嫌い、エネルギー消費を抑えようとする強力な性質、「現状維持バイアス」を持っています。「勉強が面倒くさい」と感じるのは、あなたが怠惰だからではなく、あなたの脳が正常に働いている証拠なのです。

このテンプレートの目的は、日々の浮き沈みがある「意志力」だけで脳の抵抗に立ち向かうことではありません。その抵抗を巧みにかわし、あなたの努力を無駄にしないために、韓国語学習を「歯磨き」のように、より少ないエネルギーで自動的に始められる状態へと導く「仕組み」を作ることです。

3.【報酬設計】脳に「快感」を刻み込む

目標： 脳に「韓国語学習＝良いことだ！」とインプットさせ、次もやりたくさせる。

私たちの脳は、行動の直後に「快感」を得ると、その行動を「良いことだ」と学習し、繰り返したくなります。ここでは、即座に得られる「達成感」と、少し特別な「ご褒美」の2種類を設計します。

【ワーク】あなたの脳への「ご褒美」を決めよう

（例：小さくガッツポーズをして、「よし、やった！」と心の中で言う / 手帳に「今日の私、ナイス！」と書き込む）

② ご褒美をセットにする（誘惑バンドリング）：楽しみを連結させる
「祝福のアクション」に加えて、あなたがもともとやりたいと思っていること（Want）を、韓国語学習（Should）のご褒美として設定します。これにより、学習への心理的なハードルがさらに下がります。

（例：「テキストを1ページ読んだら、少し良いコーヒーを淹れて飲む」 / 「単語を5個覚えたら、好きな音楽を1曲だけ集中して聴く」）

2.【安全ルールの設定】「ゼロにしない」ための保険

どんなに入念に計画しても、疲れていたり、時間がなかったりする日は必ず来ます。そんな時、「まあ、今日くらいはいいか」と休んでしまうと、習慣の鎖は簡単に途切れてしまいます。

そうならないために、「どんな状況でも、これだけは絶対にやる」という最低ラインを決めておきましょう。

【ワーク】あなたの「絶対的最低ライン」を決めよう

（例：アプリを1問だけ解く / 単語の音声を1分だけ聞く / テキストの単語を1つだけ眺める）

3.【週次レビュー】週末5分の作戦会議 (KPT法)

1週間が終わったら、週末に5分だけ時間をとって、感情論ではなく客観的に振り返りましょう。これにより、あなたの「習慣化の仕組み」はどんどん洗練されていきます。

【ワーク】今週の振り返りと、来週の作戦

（例：飲み会がある日は、朝のうちに最低ラインだけでもやっておく）

4. 次のステップへ - そして、その先へ

1週間の実践、本当にお疲れ様でした！もしあなたが7日間、一度も鎖を途切れさせずに実行できたなら、それはとてつもなく大きな成功体験です。

  • 負荷の調整：もし1週間が楽に感じたなら、次の週は少しだけ負荷を上げてみましょう。（例：5分 → 7分に）無理は禁物です。
  • 壁を感じたら：学習を続ける中で、必ず「伸び悩み」を感じる時期が来ます。しかし、それはあなたが成長している証拠です。苦しい時こそ、脳が新しい神経回路を必死に作っているサインだと覚えておいてください。
  • 中断しても大丈夫：もし、どうしても習慣が途切れてしまっても、絶対に自分を責めないでください。重要なのは「完璧に続けること」ではなく、「中断しても、軽やかに再開できる力（再開力）」です。「まあ、そういう日もあるよね」と呟いて、また次の日から、このテンプレートに記録することから始めればいいのです。
